Choose Your Shredder

Single Shaft Shredder

Flexible size reduction with precise control for uniform output.

  • Typical materials: Film, rigid plastic, mixed waste
  • Output size range: 20–50 mm
  • Capacity range: 300–2000 kg/h

Double Shaft Shredder

Heavy-duty shredding for bulky and rigid plastics with high torque output.

  • Typical materials: Crates, containers, bulk rigid waste
  • Output size range: 30–80 mm
  • Capacity range: 500–3000 kg/h

Pipe & Profile Shredder

Specialized shredding for long-length plastics and profiles.

  • Typical materials: Pipes, profiles, long-length plastic
  • Output size range: 40–100 mm
  • Capacity range: 400–2500 kg/h

Which Shredder Fits Your Material?

Double-Shaft

Best for:

  • Bulky rigid waste requiring high torque
  • Heavy-duty applications with large feed size

Output control: Adjustable screen size for consistent particle size

Note: Ideal for continuous operation in demanding environments

Single-Shaft

Best for:

  • Controlled particle size for downstream processing
  • Film and rigid plastic with precise size requirements

Output control: Precise screen and rotor configuration for uniform output

Note: Versatile for mixed material streams

Pipe & Profile

Best for:

  • Long-length plastic pipes and profiles
  • Specialized applications requiring length-specific handling

Output control: Optimized cutting geometry for long materials

Note: Designed for efficient processing of extended plastic forms
